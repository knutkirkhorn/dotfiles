# dotfiles

> My personal dotfiles

## Install

Run this command when installing or refreshing the dotfiles

```sh
source bootstrap.sh
```

### npm

Run this script to install some CLIs.

```sh
./npm.sh
```

### macOS

Run this script to install some apps for macOS.

```sh
./macos.sh
```
